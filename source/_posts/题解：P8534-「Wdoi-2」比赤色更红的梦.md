## 题意分析

**核心**：对于每组数据，判断 $x$ 是否 $\ge 3$ 或 $\ge 5$ 并对应地将残机 $+1$，判断 $s$ 是否 $\ge 10000000$、$\ge 20000000$、$\ge 40000000$ 或 $\ge 60000000$ 并对应地将残机 $+1$。

## 思路引导

:::warning[提问]
如何处理多组数据？
:::

:::success[成功]
使用 `while` 循环语句，每次将 $t-1$，因为 $t=0$ 时布尔值为 $0$, 即 false，所以将会跳出循环，此时循环语句正好执行 $t$ 次。格式如下：

```cpp
while(t--/*布尔表达式*/){
  //执行____
}
```
:::

:::warning[提问]
如何对 $x$ 和 $s$ 的值进行判断？
:::

:::success[成功]
使用 `if` 分支语句，格式如下：

```cpp
//普通版：
if(/*布尔表达式*/){
	//执行____
}
//压行版：
if(/*布尔表达式*/)//执行____（限一句）
```
:::

if(x>=5)ans++;
